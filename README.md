# Открытые данные PrivateSEO

Обезличенные агрегированные данные о поисковом и AI-трафике сайтов малого
и среднего бизнеса России. Обновляются автоматически каждую неделю.

- Живая страница с графиками и методикой: https://private-seo.ru/issledovaniya
- `research.json` — актуальный срез; `snapshots/` — история по датам.
- История коммитов этого репозитория — независимое подтверждение того, что
  данные не редактируются задним числом.

выгрузку из Яндекс.Метрики вживую: [контакты](https://private-seo.ru/contacts).

## Структура данных

- `research.json` — актуальный срез: `aiTraffic` (помесячные визиты из нейросетей,
  доля от органики, n сайтов), `enginesShare` (доля Google в органике по месяцам),
  `ctrByPosition` (CTR по позициям Яндекса + доля нулевых кликов)
- `snapshots/research-YYYY-MM-DD.json` — история еженедельных срезов
